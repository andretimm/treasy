## Instalação

Requerimentos:

* [NodeJS](https://nodejs.org/)


Todas as dependências são instalados com o ``Bower``

### Para instalar as dependências

``` sh
$ npm install bower -g
$ npm install http-server -g
$ npm install gulp -g
```

``` sh
$ npm install
$ bower install
```

### Criar build e executar

``` sh
$ npm start
```
