## Instalação

Requerimentos:

* [NodeJS](https://nodejs.org/)

## Lista de dependências
* [AngularJS](https://docs.angularjs.org/)

Todas as dependências são instalados com o ``Bower``

### Para instalar as dependências

``` sh
$ npm install
```

``` sh
$ bower install
```

## Criar servirdor com LiveSync

``` sh
$ gulp serve
```
